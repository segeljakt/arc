# Generics

Items can be parameterised by generic types. Generics can in addition be bounded by type class constraints.

```grammar
Generics ::= "[" [[Generic]]","+ "]"

Generic ::= [Name] [[Bounds]]?

Bounds ::= ":" [Path]"&"+
```

# Tasks

A **task** is an asynchronous function which can suspend its execution to wait for events. The `receive`, `on`, and `!` expressions can only be used inside tasks.

```grammar
Task ::= "task" [Name] [Generics]? [Params] ":" [Params] [Block]
```
